Transcriu i tradueix paraules poloneses

Escolta o llegeix la paraula en polonès i fes dues coses:

  1. Escriu com sonaria amb grafia catalana aproximada.
  2. Escriu la traducció al català.

Fes clic a Comprova o prem Retorn per verificar la resposta.

Com fer la transcripció?

  • Escriu com si fos català: no usis símbols fonètics.
  • Regles bàsiques:
    • k: - Davant a, o, u → c (kot → cot, kawa → cava) - Davant e, i → qu (kiedy → quiede, kino → quino)
    • cz → tx (czwartek → txvartek)
    • c → ts (cena → tsena)
    • sz → x (szkoła → xcoua)
    • ż i rz: - Davant ó o u → j (żółty → juute) - Davant a, e, i → z (żaba → zaba, rzecz → zec)
    • zi + vocal → j (ziemia → jemia)
    • dż → dj (dżem → djèm)
    • ś → x suau (środa → xroda)
    • si + vocal → x suau (się → xe)
    • ł → u curta (łąka → uonka)
    • j → i (język → iènzec)
    • y → e (com la e tancada de «més») Ex.: my → me, ryba → reba, mysz → mex
    • ą → on, ę → en
